## **9. Learning Objectives**
By the end of this microcredential, students will be able to:
1. **Develop Android apps** using Kotlin and Jetpack Compose.
2. **Integrate cloud services** (Firebase, Cloud Functions) into Android applications.
3. **Deploy machine learning models** using TensorFlow Lite for on-device AI.
4. **Apply open-source tools** (GitHub repositories, Google’s official SDKs) for development.

## **10. Content Description**
### **Module 1: Android Basics**
- **Topics:**
  - Kotlin fundamentals (OOP, coroutines).
  - Jetpack Compose for UI development.
- **Resources:** [Android Kotlin Guide](https://developer.android.com/kotlin), [Jetpack Compose Tutorial](https://developer.android.com/jetpack/compose/tutorial).

### **Module 2: Cloud Integration**
- **Topics:**
  - Firebase Authentication, Firestore, Cloud Functions.
  - Google Cloud SDK setup.
- **Resources:** [Firebase Docs](https://firebase.google.com/docs), [Google Cloud SDK](https://cloud.google.com/sdk).

### **Module 3: ML Integration**
- **Topics:**
  - TensorFlow Lite for on-device ML.
  - ML Kit for image recognition.
- **Resources:** [TensorFlow Lite Guide](https://www.tensorflow.org/lite/get_started), [ML Kit Docs](https://developers.google.com/ml-kit).

### **Module 4: Project Synthesis**
- **Project:** Build a full-stack Android app with:
  - Kotlin + Jetpack Compose (frontend).
  - Firebase/Firestore (backend).
  - TensorFlow Lite (ML feature).
