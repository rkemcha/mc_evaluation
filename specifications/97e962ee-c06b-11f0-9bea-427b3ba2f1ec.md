## 8. Admission Requirements:
Basic knowledge of Microsoft Excel, including navigating the interface, entering and formatting data, performing simple calculations (e.g., SUM, AVERAGE, basic arithmetic), and creating basic charts. **Learners are expected to have access to Microsoft Excel (version 2016 or newer is recommended) and a stable internet connection.**

## 9. Learning Objectives:
Upon successful completion of this microcredential, learners will be able to:
*   **Apply advanced logical, lookup, and statistical functions (e.g., nested IF, IFERROR, VLOOKUP/HLOOKUP, INDEX-MATCH, SUMIFS, COUNTIFS) to efficiently manipulate, analyze, and audit complex datasets.**
*   **Organize, clean, and validate data effectively using advanced sorting, filtering, data validation rules, conditional formatting techniques, and text manipulation tools to ensure data integrity.**
*   **Create and interpret dynamic pivot tables and pivot charts to summarize, filter, and analyze large datasets from multiple perspectives for informed decision-making.**
*   **Design and implement effective, interactive data visualizations and dashboards using various chart types to communicate insights clearly and compellingly.**
*   **Automate repetitive tasks using basic macro recording and understand the principles of efficient spreadsheet design and workflow optimization, laying a foundation for advanced automation concepts.**
*   Collaborate effectively on spreadsheets using sharing and version control features.

## 10. Content Description:
This microcredential offers an in-depth exploration of intermediate-level Microsoft Excel functionalities, empowering learners to manage, analyze, and visualize data with greater proficiency. Utilizing a blended learning model, participants will engage with self-paced online modules, interactive lab sessions, and a culminating project that simulates real-world data challenges. The curriculum is designed to be cross-curricular, providing practical skills essential for informed decision-making and enhanced productivity across various industries. **Emphasis is placed on highly demanded skills such as advanced formulas, dynamic pivot tables, interactive dashboards, and basic automation, preparing learners for data-intensive roles and fostering a data-driven mindset. All digital learning content and materials are designed to conform to WCAG 2.1 AA accessibility standards.**

*   **Module 1: Advanced Data Management & Formulas** (Approx. 25 hours)
    *   *Description*: This module builds upon foundational Excel skills by diving into more sophisticated data organization and formula application. Learners will master advanced functions, data validation, and data cleaning techniques to ensure data integrity and efficiency, which are crucial precursors to reliable data analysis.
    *   *Learning Outcomes*:
        *   Implement data validation rules to ensure data accuracy and consistency.
        *   **Apply advanced logical, lookup, and statistical functions (e.g., nested IF, IFERROR, VLOOKUP, HLOOKUP, INDEX-MATCH, SUMIFS, COUNTIFS) for complex data manipulation and analysis, including auditing formulas.**
        *   Utilize named ranges to improve formula readability, efficiency, and navigation.
        *   **Clean and transform raw data using text functions, "Text to Columns," "Remove Duplicates," and Flash Fill features to prepare datasets for analysis.**
*   **Module 2: Data Analysis & Visualization** (Approx. 25 hours)
    *   *Description*: This module focuses on extracting meaningful insights from data through powerful analysis tools and presenting them effectively using various visualization methods. Learners will master pivot tables, advanced charting, and conditional formatting to create impactful reports and interactive dashboards.
    *   *Learning Outcomes*:
        *   **Create and customize dynamic pivot tables to summarize, filter, and analyze large datasets from multiple perspectives, including using calculated fields/items.**
        *   **Develop interactive pivot charts and standalone charts to visualize pivot table data and communicate key trends and insights effectively.**
        *   **Apply advanced conditional formatting rules (e.g., icon sets, data bars, color scales, custom formulas) to visually emphasize specific data patterns, anomalies, or thresholds.**
        *   **Design and implement effective data visualizations using various chart types (e.g., scatter plots, combo charts, sparklines, geographical maps, waterfall charts) for compelling data storytelling.**
        *   **Utilize "What-If Analysis" tools (Goal Seek, Scenario Manager, Data Tables) for basic data forecasting, sensitivity analysis, and decision support.**
        *   **Implement slicers and timelines to create interactive dashboards for dynamic data exploration.**
*   **Module 3: Automation & Collaboration** (Approx. 25 hours)
    *   *Description*: This module introduces learners to tools for enhancing productivity through basic automation and effective collaboration. It covers macro recording, efficient spreadsheet design principles, and features for sharing and version control, preparing learners for more efficient and collaborative work environments.
    *   *Learning Outcomes*:
        *   **Record and execute basic macros to automate repetitive tasks and improve workflow efficiency.**
        *   **Apply principles of efficient spreadsheet design for clarity, maintainability, and optimal performance.**
        *   **Understand fundamental concepts that lay the groundwork for more advanced automation (e.g., VBA, Python in Excel).**
        *   **Use Excel's sharing and co-authoring features to collaborate effectively on workbooks.**
        *   **Manage workbook versions and track changes to maintain data integrity and accountability in collaborative projects.**

**This microcredential aligns with key competencies from frameworks such as:**

## 15. Learning Activities:
*   **Blended Learning**: Asynchronous video lectures, interactive tutorials, readings, and case studies. Synchronous online Q&A sessions, guided lab support, and group discussions.
*   **Active Learning**: Hands-on lab exercises with step-by-step instructions, realistic practice datasets (including "messy" data for cleaning), online quizzes, and peer review sessions for chart designs and data storytelling.
*   **Project-Based Learning**: Module-level practical assignments and a comprehensive final project requiring integrated application of skills to real-world scenarios.
*   **All learning materials will be provided in accessible digital formats, and accommodations for learners with disabilities will be made in accordance with institutional accessibility policies.**

## 16. Workload Theoretical:
Approximately 25 hours (comprising video lectures, readings, online tutorials, and conceptual discussions).

## 17. Workload Practical:
Approximately 50 hours (comprising hands-on lab exercises, independent practice with datasets, module assignments, and the final project).

## 18. Assessment Methods:
*   **Formative Assessments**: Short online quizzes (multiple-choice, fill-in-the-blank) and small practical exercises within modules to provide ongoing feedback and check understanding.
*   **Summative Assessments**:
    *   **Module 1 Practical Assignment (20% of total grade)**: Learners clean a raw dataset, apply advanced formulas (including nested functions), and implement data validation rules to answer specific business questions.
    *   **Module 2 Practical Assignment (20% of total grade)**: Learners analyze a dataset using pivot tables, create an interactive dashboard with appropriate charts, and apply advanced conditional formatting to highlight key insights.
    *   **Final Project (60% of total grade)**: A comprehensive, project-based assessment where learners apply all acquired skills (data cleaning/validation, advanced formulas, pivot table analysis, interactive dashboard creation, basic macro recording) to a realistic, complex dataset, culminating in a brief analytical report.
*   **All summative assessments will be evaluated using detailed, transparent rubrics that clearly outline criteria for success. Guidelines for grading consistency across instructors will be established to ensure fairness and reliability.**
